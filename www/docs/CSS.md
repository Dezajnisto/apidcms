# CSS-стили — управление и кэширование

## Редактор CSS

Админка → **Дизайн → CSS-стили**.

CSS-стили хранятся в файле `storage/css/custom.css` в корне проекта.
Редактировать можно прямо в админке — редактор с подсветкой, Tab → 4 пробела, Ctrl + S → сохранить.

## Кэширование браузеров (cache-busting)

При сохранении CSS через админку автоматически инкрементится версия файла.
В base.html.twig CSS подключается с параметром `?v=N`:

```
<link rel="stylesheet" href="/storage/css/custom.css?v=2">
```

Браузеры видят новый URL и загружают свежий CSS вместо закэшированного.

Ничего делать не нужно — версия растёт сама при каждом сохранении CSS.
Если нужно принудительно сбросить кэш — просто открой CSS-редактор и нажми «Сохранить» (даже без изменений).
